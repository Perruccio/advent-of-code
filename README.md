# Advent of code
Learning Python by solving Advent of Code.

## Installation
In project folder run `pip install -e .` to install the package (or `pypy -m pip install -e .` with Pypy).

## Tests
To run all tests just do `pytest` (or `pypy -m pytest` with Pypy) in project folder.
Run `pytest solutions/yearYYYY` for a specfic year or `pytest solutions/yearYYYY/dayDD` for a specific day.
